\section*{Solution}

To solve the equation \textbackslash{}$2x + 3 = 7\textbackslash{}$, we subtract 3 from both sides:

\begin{equation}
2x = 4
\end{equation}

Then divide both sides by 2:

\begin{equation}
x = 2
\end{equation}

Therefore, the answer is \textbackslash{}$x = 2\textbackslash{}$.
